You can install Java Runtime Environments (JREs) available from the Install JRE dialog, and customize base image, according to your requirements. The JRE version is listed in the Select JRE to install drop-down list. The corresponding configurations are listed in the Select a Configuration for Java 8 Update 91 drop-down list. The Next button gets enabled when either the Selected Desktop or All Filtered Desktops option is selected.
